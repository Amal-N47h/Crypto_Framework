## Creating the Issues on GitHub

Run the helper script from the repository root to bulk-create all 10 issues automatically:

```bash
export GITHUB_TOKEN=<your_personal_access_token>
python create_issues.py
```

The token needs **repo** scope (Issues: Read & Write).  
Create one at <https://github.com/settings/tokens>.

Alternatively, each markdown file can be copy-pasted manually into  
**GitHub → Issues → New Issue**.
